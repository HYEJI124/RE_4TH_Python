HyperText

  1. 웹 페이지에서 다른 페이지로 연결되는 링크를 포함하는 텍스트
  2. 비선형적 정보 접근 가능
  3. 예: 위키피디아의 파란색 링크들

MarkUp

  • 태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어
  • 표시(Mark)를 한다(Up)는 의미
  • 문서의 구조와 의미를 표현
