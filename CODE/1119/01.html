나의 첫 css 페이지

HTML 구조 + CSS 스타일 실습

CSS란 무엇인가?

CSS(스타일 시트)는 HTML 문서의 시각적 표현을 담당합니다. 색상, 크기, 간격, 레이아웃 등을 정의할 수 있으며 웹 페이지를 더욱 사용하기 쉽게 만들어 줍니다.

학습 내용

  • CSS 기본 선택자와 우선순위
  • 폰트와 색상 설정
  • 마진, 패딩, 박스 모델
  • Flexbox와 Grid 기초

CSS 팁

  • class는 여러 번 사용 가능/id는 한 번만 사용 가능
  • box-sizing: border-box 추천
  • flexbox로 정렬 문제 쉽게 해결
  • 반응형 디자인을 위해 rem 단위 사용 추천
